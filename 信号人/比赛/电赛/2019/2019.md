# 简易电路特性测试仪

![alt text](image.png)

### 基本要求
1. 输入电阻
2. 输出电阻
3. 增益
4. ==幅频特性==


### 进阶要求
1. 电阻 开路或短路
2. 电容 开路
3. 电容 两倍
4. 判断速度 < 2s


现象：
