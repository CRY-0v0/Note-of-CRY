# 绪论
- 不是高频信号，根本不用在意这么多，直接连连看就行了
- 对于敏感的信号，要做好地隔离，以及远离功率区域。

# 方案
|第n层  |结构    |方案一     |方案一  |方案一
|:---:  |:---:   |:---:     |:---:  |:---:
|    |Silkscreen    |       |       |
|    |Soldermask    |       |       |
|L1  |Copper        |TOP    |TOP    |GND1
|    |Prepreg       |       |       |
|L2  |Copper        |GND1   |PWR1   |S1
|    |Core          |       |       |
|L3  |Copper        |PWR1   |GND1   |S2
|    |Prepreg       |       |       |
|L4  |Copper        |BOTTOM |BOTTOM |PWR1
|    |Soldermask    |       |       |
|    |Silkscreen    |       |       |

```
# 方案一：
    常见PCB方案
# 方案二：
    适用于主要元器件在底层布局，或关键信号底层布线的情况
# 方案三：
    适用于元器件一插件为主的PCB，常常考虑电源在布线层S2中实现，底层为地平面，进而构成屏蔽腔体
```
