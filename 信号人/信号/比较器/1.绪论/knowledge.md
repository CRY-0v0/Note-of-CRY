# 一、基本参数

- $U_{OH}$ 输出电压高值
- $U_{OL}$ 输出电压低值
- $U_{t}$  跃变阈值电压

# 二、电路分类与特点

1. 单限比较器

- 单个$U_{t}$
- 反应灵敏（不稳定，抗干扰弱）
- 速度快

2. 滞回（迟滞）比较器

- 两个受输出状态控制的$U_{t}$

3. 窗口比较器
