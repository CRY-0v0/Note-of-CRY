# <center>gitignore

功能：
- 用于忽视推送文件

方法：
- 创建一个文件：.gitignore
- 在该文件内输入相关语句

语句：

|语句       |作用 
|:---:      |:---:|
|#       	|注释
|test	    |忽略所有包含test的文件，文件夹
|/test	    |从项目根目录开始匹配，忽略名为test的文件夹（非文件）
|test.*     |忽略所有以test为 基本名 的文件
|path/test.*|忽略path路径下，以test为 基本名 的文件
|!       	|取消忽略，即使之前有忽略规则（常用于，出现多个规则文件时）
|       	|

> /路径/基本名.拓展名
